# News Upload App 📰

## Fitur
- ✅ Upload berita dengan judul, konten, dan gambar
- ✅ Preview gambar sebelum upload
- ✅ UI modern dengan glassmorphism & animasi
- ✅ Responsive design
- ✅ Data tersimpan di LocalStorage
- ✅ Daftar berita yang sudah diupload

## Cara Menjalankan
1. Buka `index.html` di browser
2. Isi form upload berita
3. Pilih gambar dan klik "Upload Berita"
4. Lihat hasil di bagian bawah

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- LocalStorage
